# FSharp Scripts Repository

This repository contains **F# scripts for learning and automation**, including:

- `DirList.fsx` – list directories and files
- `stat.fsx` – show file and directory metadata (size, last modified, permissions)
- Other F# scripts for file management and experimentation

## How to Run

Ensure you have **.NET SDK** installed. Then run:

```bash
dotnet fsi stat.fsx <file-or-directory>
dotnet fsi DirList.fsx <directory>
